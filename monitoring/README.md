## План разработки веб-интерфейса
1. UI/UX: карта экранов (дашборд, ноды, процессы, контейнеры, порты, логи, алерты, отчеты), сценарии навигации, адаптивность.
2. API: спецификация REST/WS контрактов для всех виджетов, описание форматов ошибок и аутентификации.
3. Архитектура фронта: структура PHP-страниц, layout (sidebar/header), роутинг, инклюды общих компонентов.
4. Стили: базовый дизайн-система (цвета, типографика, сетки), тёмная тема, UI-стейт (загрузка, ошибки, пустые).
5. Дашборд: карточки KPI, графики CPU/RAM/диск/сеть, живые обновления через WebSocket.
6. Ноды: таблицы, CRUD-формы, статусы, фильтры, модалки добавления/редактирования, массовые операции.
7. Процессы/контейнеры: real-time списки, управление действиями (start/stop/kill), логи, метрики, поиск.
8. Порты/сеть: визуализация открытых портов, управление firewall, диагностика подключений.
9. Логи/алерты: централизованный просмотр, фильтры, экспорт, настройка порогов уведомлений.
10. Доп. инструменты: файловый менеджер, веб-терминал, настройки пользователей/ролей, i18n.
11. Тесты и сборка: e2e-скрипты, линтеры, GitHub Actions, подготовка prod-конфига.
