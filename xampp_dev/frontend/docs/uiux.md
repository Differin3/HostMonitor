# UI/UX план
1. Навигация: sidebar + top header, breadcrumb, быстрый поиск.
2. Экраны:
   - Дашборд: KPI карточки, системные графики, алерты.
   - Ноды: таблица, health badge, фильтры, детали в выезжающей панели.
   - Процессы: real-time список, действия (start/stop/kill).
   - Контейнеры: статусы, ресурсы, логи, кнопки управления.
   - Порты/сеть: визуализация портов, firewall, правила.
   - Логи/алерты: timeline, фильтры по уровню, экспорт.
   - Настройки: пользователи, роли, SSH ключи, темы.
3. Макет: двухколоночный, гибкая сетка карточек, breakpoint 1280/992/768.
4. Тема: светлая по умолчанию, переключатель dark, единая палитра (#1f2937, #2563eb, #10b981).
5. UI-состояния: loading skeletons, empty states с CTA, inline errors.
6. UX-детали: модалки с подтверждением, notifs toast, drag-drop для правил, autorefresh WebSocket.
